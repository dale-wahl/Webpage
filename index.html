2017 SAT versus ACT Participation Rates

Exploratory data analysis on SAT and ACT scores in 2017.

I combined data on both SAT and ACT scores for the 2017 school year and compared them. The focus was a presentation with recommendations for the SAT college board to improve their participation rates. You can find the code and data available here, if you would like to look at the process or work with it yourself.

  • Date: October 2017
  • Category: EDA

Iowa Liquor Sales

Using open data for market research

In this project I took data from data.iowa.gov as well as from the US Census in order to look at purchases of liquor stores in Iowa. I then used that data to predict future purchases and identified the cities where the best opportunities for new stores existed. You can read more about the project on my blog here. You are also free to look at my presentation as well as the actual code on my github page here.

  • Date: October 2017
  • Category: Linear Regression
