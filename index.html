West Nile Virus

Predicting outbreaks in Chicago

I worked on a group project for a Kaggle contest that was meant to predict whether or not a mosquito trap would contain those with the West Nile Virus. The two key components we looked at were weather data within the past nine months and spacial data. We also looked at the effects of spraying with a limited set of information on some historical sprays. Our model was able to predict 81% of actual cases of WNV based on our test data. Find the code here and the presentation here!

  • Date: November 2017
  • Category: Geospacial Modeling, XGBoost Model, Confusion Matrix

Conde Nast Twitter Engagement

How do we measure and predict Twitter engagement?

We were asked to explore how users were interacting with different brands on Twitter. Useing natural language processing on the tweet text and the link headlines, we found topics that were more or less likely to lead to retweets or favorites. We also utilized Amazon's Rekognition image processing to convert photos to descriptions and identify aspects such as different celebrities. We then claffified the images into different groups with LDA modeling and compared and contrasted those categories to find ones that were most likely to drive engagement. You can find the presentation here. The code is, of course, theirs.

  • Date: December 2017
  • Client: TrackMaven
  • Category: Natural Language Processing, Amazon Rekognition

Telsa Stock Price Prediction

Can following Elon Musks Tweets Help Us Predict TSLA Prices?

To answer the above: Kinda. I took on this project to teach myself how to use ARIMA modeling and handle time series data (which is one of the more tricky data science problems). Predicting stock prices is notoriously tricky, but when you have a hypothosis, data science is an excellent way to test it! Mine in this case was that Telsa is mostly a speculative stock and thus would be heavily influenced by current news. You can read a blog post I made about ARIMA modeling for this project here. You can also find more code including handling Twitter API, scraping news sites, model building, and combining it all here. This is an ongoing project as I incorporate more sources to make improvements.

  • Date: December 2017
  • Category: Time Series Modeling, Natural Language Processing
